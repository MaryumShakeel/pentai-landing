# PentAi Landing Page

Dark-themed AI financial platform landing page built with **Next.js 14** and **Tailwind CSS**.

## Sections
- Navbar (responsive with mobile menu)
- Hero with CTA
- Our Supporters
- Financial Dashboard Preview
- Stats (23% cards)
- Product Features
- Comparison Section
- Feature List + Testimonials
- CTA Banner
- Footer

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy on Vercel

### Option 1 - Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2 - GitHub + Vercel Dashboard
1. Push this project to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "New Project"
4. Import your GitHub repo
5. Click "Deploy" — done!

## Build
```bash
npm run build
npm start
```
